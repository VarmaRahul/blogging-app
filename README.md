# 📝 Blogging App

A full-stack blogging platform that allows users to create, read, update, and delete blog posts with authentication and a modern UI.

---

## 🚀 Features

- 🔐 User Authentication (Sign Up / Login / Logout)
- 📝 Create, Edit, Delete Blog Posts
- 📖 Read Blogs with Rich Content
- 💬 Comment System (optional if implemented)
- 🎨 Responsive UI
- ⚡ Fast and modern frontend using React
- ☁️ Backend powered by Appwrite

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5, CSS3, JavaScript
- Tailwind CSS / CSS Modules (if used)

### Backend
- Appwrite (BaaS)

### Database
- Appwrite Database

### Authentication
- Appwrite Auth
